# CS490 Fitness App Backend

This is the backend for the CS490 Fitness App, built with Flask and SQLAlchemy.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Set up environment variables:
   - Create a `.env` file in the root directory.
   - Add your database URL: `DATABASE_URL=mysql://user:password@host:port/database`
   - Ensure you have the `ca.pem` file for SSL connection if using MySQL.

3. Run the app:
   ```
   python app.py
   ```
